# 📊 Baskin Robbins Retail Sales Analysis Report

## 📌 Project Overview
This project is a Retail Sales Analysis Dashboard built using Power BI.  
The dashboard provides insights into sales performance, store performance, payment trends, and monthly revenue analysis.

## 📂 Dataset Fields Used
- payment summary
- Order_count
- Total_qty
- avg_payment_summary
- dis_count_item
- dis_count_category
- Category
- Store Name
- Payment Mode
- Month
  
## 📊 Dashboard Features
- Total Payment Summary (KPI Card)
- Total Orders (KPI Card)
- Total Quantity Sold (KPI Card)
- Average Payment Summary (KPI Card)
- Discount Count by Item (KPI Card)
- Discount Count by Category (KPI Card)
- Payment Summary by Category (Bar Chart)
- Payment Summary by Store Name (Column Chart)
- Payment Summary by Payment Mode (Donut Chart)
- Monthly Payment Trend (Line Chart)
- Interactive Slicers (Category, Payment Mode, Store Name)

## 🛠 Tools Used
- Power BI
- DAX (Measures & Calculations)
- Data Visualization
- 
## 📈 Key Insights
- Identified top-performing store locations
- Analyzed revenue contribution by category
- Compared payment mode distribution
- Evaluated monthly sales trends
- Tracked discount impact on sales
